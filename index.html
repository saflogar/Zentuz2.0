Zentuz Documentation
Zentuz Kinect Framework(ZKF) was developed to simplify the communication in both ways, for displaying the kinect output and for receiving input by using Windows Presentation Foundation. It was written in c# and the full source code can be found in https://github.com/saflogar/Zentuz/tree/master/Framework. Inside the framework folder we will find 7 classes:
  • CursorAdorner
  • GesturePoint
  • HoverButton
  • KinectButton
  • KinectCursorEventArgs
  • KinectCursorManager
  • KinectInput

How does it works?

The core of the ZKF is the KinectCursorManager which coordinates all the hand tracking logic and raise the correct events according to each situation. The KinectCursorManager get the position of all the human bodies who are in front of the kinect, once it has detected all the bodies, it draw a circle in the hand of the user that is nearer to the kinect. After getting the nearest body, the GetPrimaryHandMethod receive this skeleton and find the nearer hand of this body.
